Solar Panel Monitoring System

A system to monitor and optimize solar panel performance.

Overview

This project involves developing a solar panel monitoring system using Python and ESP32 microcontrollers. The system collects data on solar panel performance, including voltage, current, and temperature, and sends this data to a web interface for real-time monitoring and analysis. The goal is to optimize energy production and identify potential issues with the panels. What the project does, why you built it, and what problem it solves.

Technologies Used

  • ESP32
  • HTML/CSS
